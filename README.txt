## Installation

1. Ensure you have Python installed on your system.

2. Install the `psycopg2` package using pip.

3. Set up a PostgreSQL database.

4. Update variables in `connect_to_database()` function to match that of your database.

## Usage

1. Open a terminal in the directory containing `COMP3005-A3-Q1.py`.

2. Run the main Python script: `Python3 COMP3005-A3-Q1.py`

3. Follow the on-screen instructions to perform CRUD operations on the student records:
- Enter the corresponding number for the action you want to perform.
- Follow any prompts to input data (ex: student details for adding a new student).
